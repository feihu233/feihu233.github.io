三角函数

2020-06-23 数学

反三角函数: 三角函数的反函数
arcsin(x) 定义域 [-PI/2, PI/2], 值域 [-1, 1]
arccos(x) 定义域 [0, PI], 值域 [-1, 1]
arctan(x) 定义域 (-PI/2, PI/2), 值域 (-∞, +∞)
arccot(x) 定义域 (0, PI), 值域 (-∞, +∞)

少见表达方式
sec(x) = 1 / cos(x)
csc(x) = 1 / sin(x)

六边形法则
sin(x) cos(x)
tan(x) 1 cot(x)
1/cos(x) 1/sin(x)
平方和: 倒三角部分的两上端点平方和等于下端点平方
sin^2(x) + cos^2(x) = 1
tan^2(x) + 1 = 1/cos^2(x)
cot^2(x) + 1 = 1/sin^2(x)
任一边点元素 = 顺时针相邻两点比值 = 临近两点乘积
sin(x) = cos(x) / cot(x) = tan(x) * cos(x)

万能替换公式 -> 替换为 tan(x/2)
six(x) = (2 * tan(x/2)) / (1 + tan^2(x/2))
cos(x) = (1 - tan^2(x/2)) / (1 + tan^2(x/2))
tan(x) = (2 * tan(x/2)) / (1 - tan^2(x/2))

其他公式
arctan(x) + arccot(x) = PI/2
tan(a+b) = (tan(a) + tan(b)) / (1- (tan(a) * tan(b)))
